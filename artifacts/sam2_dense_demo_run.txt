WHAT THIS IS
------------
The no-download companion to panoptic_annotate_video.py. It runs SAM 2's automatic mask
generator over a grid of point prompts, so it needs no text prompt and no new weights —
only the sam2.1_hiera_large.pt checkpoint already in checkpoints/sam2/.

WHAT THIS IS NOT
----------------
SAM 2 is a segmentation model, not a panoptic one. It finds object boundaries but has no
class vocabulary, so regions are labelled "region #N" — never "road" or "car #3" — and
there is no thing/stuff split. For real semantic labels use panoptic_annotate_video.py.

  Model      : SAM 2 automatic mask generator (SAM2AutomaticMaskGenerator)
  Checkpoint : checkpoints/sam2/sam2.1_hiera_large.pt  (local, pre-existing)
  Config     : configs/sam2.1/sam2.1_hiera_l.yaml      (hydra name, resolved via sam2 pkg)

Deliberately not carried over, with reasons:

  BOX_THRESHOLD=0.35      Grounding DINO knobs. This script has no text prompt and no box
  TEXT_THRESHOLD=0.25     detector, so they have no target here. --score-threshold is SAM
                          2's pred_iou_thresh (mask quality) — a different quantity, not a
                          rename of BOX_THRESHOLD.
  PROMPT_TYPE_FOR_VIDEO   No prompts exist; regions come from a 24x24 point grid.
  multimask_output=False  Correct for the reference (one mask per box prompt), harmful
                          here. Measured on frame 0 of the KITTI clip:
                              multimask_output=True   21 regions, 84.3% coverage
                              multimask_output=False  15 regions, 62.1% coverage
                          A 22-point coverage loss defeats the point of a dense script, so
                          the default stays True. Pass --no-multimask-output to match the
                          reference exactly.

Also not copied: the reference's global torch.autocast(...).__enter__(), which stays
entered for the life of the process. This script scopes autocast to the inference call —
same dtype, no leak.

WHY BOTH SCRIPTS EXIST
----------------------
SAM 2 covers ~80% of pixels with no semantics at 1.22 fps; Mask2Former covers 100% with
full Cityscapes labels at 11.15 fps — roughly 9x faster and strictly more informative.
panoptic_annotate_video.py is the right tool for dense annotation; this script is the
fallback for environments where downloading panoptic weights is not permitted.
